This is a very simple but useful plugin, provides a sidebar widget to show Top Rated Events.



== Description ==
This plugin can be used by users who are already using the plugins, Events Manager (by Marcus Sykes) and kk Star Ratings (by Kamal Khan).
This plugin shows highest rated events with events images, events details, events star ratings in a sidebar widget. As an optional feature this plugin can show social share icons if Social Sharing Toolkit (by LinksAlpha) plugin is installed.
I developed this plugin to one of my projects, and I thought of sharing this with other developers and wordpress lovers.


== Installation ==



1. Upload the folder 'top-rated-events' found inside the zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. No extra settings.

Note : You should install Events Manager plugin and kk Star Rating plugin, before installing this plugin.
